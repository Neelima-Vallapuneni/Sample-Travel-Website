# Travel Website

This is a personal and responsive **Travel Website** built using **HTML** and **CSS**. It is designed to showcase travel destinations, tour packages, and services with a clean and attractive layout.

## Features
- Responsive design for all devices  
- Home section with background image  
- About Us section  
- Tour packages section  
- Services section  
- Contact section  
- Smooth scrolling  
- Simple hover animations  

## Technologies Used
- HTML  
- CSS
